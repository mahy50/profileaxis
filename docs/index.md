# AI 铝型材网页设计系统

## 项目概述

**目标：** 打造一个 AI 驱动的 3D 铝型材网页装配设计器，核心定位为"受约束的会话式装配设计器"，而非通用 Web CAD。

**产品主线：**
自然语言意图 → 确认层 → 结构草案 DSL → 规则合法化 → 前端本地参数化建模 → 设计 BOM → 交易 BOM / 采购映射 → PDF 方案单

**首版边界（已冻结）：**
- ✅ 矩形、正交、开放式、落地式储物架 / 置物架
- ✅ 自然语言生成 + 局部编辑
- ✅ 设计 BOM + 交易 BOM + PDF 导出
- ❌ 自由建模 / 复杂仿真 / 实时价格 / 账号体系

**技术栈：**
- 前端：Vue 3 + TypeScript + Vite + Pinia + Babylon.js
- 后端：Node.js（AI 编排、PDF 导出、catalog ETL）
- 架构：monorepo（apps + packages + tools）

**核心理念：**
- `resolvedDsl` 是唯一真相源，3D 只是投影
- AI 只出草案，规则层兜底合法化
- 前端本地建模为主，后端只做 AI 编排和正式导出
- 规格先于 SKU，设计期不绑定商城
